# Collectedge

Modern landing site for the Collectedge product.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Project Structure
- `index.html` - App entry HTML
- `src/App.tsx` - App composition
- `src/components/` - UI sections
- `src/assets/` - Images and static assets

## Notes
- Built as a static marketing site.
- Styling is handled with Tailwind utility classes.
